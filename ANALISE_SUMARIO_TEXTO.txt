================================================================================
O QUE O PROJETO FAZ
================================================================================

Landing Page:
  - Website moderno com design responsivo
  - Hero section com call-to-action (WhatsApp)
  - Seções de soluções e FAQ interativo
  - Blog preview (últimos 3 posts)
  - Otimizado para SEO e performance

Sistema de Blog:
  - 3 posts Markdown (Materiais, Guia, DIY)
  - Listagem com filtros por categoria
  - Paginação (9 posts por página)
  - Posts relacionados automáticos
  - Meta tags SEO em cada post
  - RSS feed e Sitemap automáticos

Painel Administrativo:
  - Login com token GitHub
  - Editor Markdown WYSIWYG
  - CRUD completo de posts
  - Upload de imagens
  - Gerenciador de autores
  - Integração com GitHub API

Automação:
  - GitHub Actions para build
  - Validação automática de posts
  - Geração de HTML estático
  - Deploy automático via GitHub Pages

================================================================================
COMO FUNCIONA O SISTEMA
================================================================================

1. VISUALIZAR SITE (User)
   ├─ Acessa conexaoterrabambu.com.br
   ├─ GitHub Pages serve arquivos estáticos
   ├─ Carrega HTML + CSS + JS
   ├─ JS carrega posts do posts.json
   └─ Site interativo em <2 segundos

2. CRIAR POST (Admin)
   ├─ Admin acessa admin/index.html
   ├─ Faz login com token GitHub
   ├─ Preenche formulário (Markdown)
   ├─ Clica "Salvar"
   ├─ JavaScript valida campos
   ├─ Envia para GitHub API
   ├─ GitHub recebe arquivo
   ├─ Webhook dispara GitHub Actions
   ├─ Actions executa npm run build
   ├─ Gera blog/slug/index.html
   ├─ Atualiza posts.json
   ├─ Atualiza sitemap.xml e rss.xml
   ├─ GitHub Pages publica
   └─ Post está LIVE em blog/slug/

3. FLUXO DE SEO
   ├─ Cada post tem meta tags
   ├─ Open Graph para redes sociais
   ├─ Sitemap atualizado automaticamente
   ├─ RSS feed para leitores
   └─ Google indexa automaticamente
