# SPEC: Site Institucional

**Domínio:** Branding e Identidade Visual  
**Tecnologia:** HTML/CSS/JS Puro  
**Caminho:** `/` (Raiz)

---

## 1. Objetivo Principal

- Apresentar a **Conexão Terra Bambu** como a marca mãe do ecossistema.
- Direcionar o tráfego para os funis específicos (Forros, Consultoria, Educação).
- Consolidar a autoridade através de portfólios e visão institucional.

## 2. Componentes da Home (`index.html`)

- **Hero Vídeo**: Impacto emocional com foco no "Imagine seu espaço ideal".
- **Especialidades**: Grade completa de serviços (Taipa, Bambu, Madeira, Tintas).
- **Link para LP**: Botão central direcionando para o funil de Forros (`lp/forros-bambu/`).

## 3. Gestão Legada

- A home atual utiliza HTML puro para máxima velocidade de carregamento (SEO técnico).
- Manter links de WhatsApp consistentes com o funil principal.

## 4. Evolução

- No futuro, a Home pode ser integrada à tecnologia React para facilitar a gestão centralizada via `.agents`.
