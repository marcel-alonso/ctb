Autenticação via GitHub

Como obter seu token:
  1. Acesse GitHub Settings → Developer settings → Personal access tokens
  2. Clique em Generate new token (classic)
  3. Dê um nome descritivo (ex: "CTB Admin")
  4. Selecione escopos: repo (acesso completo aos repositórios privados)
  5. Gere o token e copie-o aqui
  6. ⚠️ Revogue imediatamente qualquer token que foi exposto publicamente
